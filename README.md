###*Bem vindo a Powerpedia!*

_Powerpedia_ é um projeto pessoal para facilicar a criação de personagens e consulta de textos ao jogar o RPG de Mesa da Jambo Editora 3D&T Victory!
Feito de fã para fãs o projeto foi uma forma para estudo de ferramentas como react e desenvolvimento pessoal como também para uma forma de ajudar os jogadores.

Tecnologias usadas:
-React
-SASS
-Bootstrap
-Supabase / PostgreSQL
